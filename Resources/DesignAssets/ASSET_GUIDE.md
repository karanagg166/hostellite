Place custom design images here.

Suggested use:
- Put large onboarding illustrations in `Illustrations/`
- Put custom icons or logos in `Icons/`
- After you add the real files, we can move the final assets into `Assets.xcassets` or load them directly

You do not need to provide every PNG/JPEG right away.

Helpful first files to provide for exact UI matching:
- The 2 large onboarding/hostel illustration images
- Any custom logo mark that should replace the current vector `H`
- Any non-standard icons that must exactly match the design

You usually do not need to provide:
- Apple / Google / Facebook icons
- Basic arrows
- Simple status icons

Those can stay as SF Symbols or vector shapes unless you want pixel-perfect matching.
